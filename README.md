## Description

This is my boilerplate to use html and css project

### NPM Modules

node-sass
Concat
Autoprefixer
postcss-cli
npm-run-all

## Setup

### Install

```bash
git clone git@github.com:mucahidyazar/nexter-project
cd nexter-project
npm i or yarn
npm i -g npm-run-all node-sass
npm run start
```

### Production

Open the index.html to view the app.
